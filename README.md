# Kvasir-Dataset

The Kvasir Dataset is a publicly released endoscopic image data set of gastrointestinal tract.In total, the dataset contains 8000 images that belongs to 8 different classes of diseases catergorize under anatomical landmarks and patholigical and normal findings.

The dataset can be downloaded via : https://datasets.simula.no/kvasir
